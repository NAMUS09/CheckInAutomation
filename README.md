## Prerequisites

- Python 3.11
- Selenium library (`pip install selenium`)

## Getting Started

### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/your-username/check-in-automation.git
    cd check-in-automation
    ```

2. Install dependencies:

    ```bash
    pip install selenium
    ```

### Usage

Run the application:

```bash
python main_app.py
